🚀 HOW TO USE:

1. OPEN THE APPLICATION:
   → http://localhost:8080

2. DASHBOARD TAB:
   → Click "Dashboard" to view real-time statistics
   → See total employees, check-ins, and who's currently working

3. ADD EMPLOYEES:
   → Click "Add Employee" tab
   → Enter Employee ID and Name
   → Click "Add Employee" button

4. CHECK IN/OUT:
   → Click "Check In/Out" tab
   → Enter Employee ID
   → Click "✓ Check In" to mark arrival
   → Click "✗ Check Out" to mark departure

5. VIEW HISTORY:
   → In "Check In/Out" tab
   → Enter Employee ID
   → Click "View History"
   → See all check-in/check-out records

6. GENERATE REPORTS:
   → Click "Reports" tab
   → Select date range
   → Click "Generate Report"
   → View attendance summary

🔗 API EXAMPLES:

Add Employee:
  curl -X POST http://localhost:8080/api/employee \
    -H "Content-Type: application/json" \
    -d '{"id": 5, "name": "Eve Taylor"}'

Check In:
  curl -X POST http://localhost:8080/api/checkin \
    -H "Content-Type: application/json" \
    -d '{"employeeId": 5}'

Check Out:
  curl -X POST http://localhost:8080/api/checkout \
    -H "Content-Type: application/json" \
    -d '{"employeeId": 5}'

Get Attendance:
  curl "http://localhost:8080/api/attendance?employeeId=5"

Get Report:
  curl "http://localhost:8080/api/attendance/range?from=2026-05-27T00:00&to=2026-05-28T00:00"
